Current release: LAST_VERSION_TAG.

The releases below contain the Pydev Extensions bundled with the Pydev "Open Source" version.

  • Use the Eclipse update manager: http://www.pydev.com.br/updates
  • Get Latest release as a zip: pydev_extensions_LAST_VERSION_TAG.zip
  • Older releases may be found at: http://www.pydev.com.br/zips

NOTE: If you already have the matching Open Source version installed, it will not appear in the Eclipse update manager.


Requirements:

  • Eclipse 3.1 (the SDK is required)
  • Python and/or Jython
  • Java 1.4 (reccomended) or higher

Important:

The Pydev Extensions version must always be accompanied by the same version of the Pydev "Open Source" version.



Instructions:

Update Manager

Go to the update manager (help > Software Updates > Find and Install > Search for new feature to install), add: http://www.pydev.com.br/updates/ as a 'remote site' and let Eclipse 'finish' it for you.


Zip file

After downloading the zip file, you have to extract it yourself on top of Eclipse. If you choose to do it, just make sure the plugins folder is extracted on top of the Eclipse plugins folder. After that, restart it with the '-clean' flag, to make sure Eclipse finds out about it.


Important: Eclipse must be installed in a path that does not have any spaces.
