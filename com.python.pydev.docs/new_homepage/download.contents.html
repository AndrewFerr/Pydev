The referenced places below contain the Pydev Extensions and the Pydev "Open Source" version.

  • Use the Eclipse update manager: http://www.fabioz.com/pydev/updates
  • Get zip releases: http://www.fabioz.com/pydev/zips

NOTE: If you already have the matching Open Source version installed, it will not appear in the Eclipse update manager.


Requirements:

  • Eclipse 3.2 (the SDK is required)
  • Python and/or Jython
  • Java 1.4 (reccomended) or higher

Important:

The Pydev Extensions version must always be accompanied by the same version of the Pydev "Open Source" version.

Pydev is tested against Python 2.4 and Jython 2.1, but it *should* work earlier or later versions (please report a bug if it doesn't).


'Quick Install':

Update Manager

Go to the update manager (help > Software Updates > Find and Install > Search for new feature to install), add: http://www.fabioz.com/pydev/updates/ as a 'remote site' and let Eclipse 'finish' it for you.


Zip file

After downloading the zip file, you have to extract it yourself on top of Eclipse. If you choose to do it, just make sure the plugins folder is extracted on top of the Eclipse plugins folder. After that, restart it with the '-clean' flag, to make sure Eclipse finds out about it.


Important: Eclipse must be installed in a path that does not have any spaces.


'Complete Install':

A complete step-by-step on how to install pydev can be found at the Getting Started session of the manual.
