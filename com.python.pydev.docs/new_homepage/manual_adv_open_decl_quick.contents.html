The 'Open Declaration Quick Outline' allows you to quickly see (and go to) all the available 'declaration' tokens, including:

  • Class declarations
  • Method declarations
  • Global variables
  • Class and instance attributes

KEYBINDING: Ctrl+Shift+T

NOTE: Only available in Pydev Extensions
