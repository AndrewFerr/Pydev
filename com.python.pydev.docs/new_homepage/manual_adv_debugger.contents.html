The debugger is available for both, the Pydev 'Open Source' version and the Pydev Extensions, however, the Pydev Extensions plugin allows you to debug from a program that is not launched from within Eclipse.

This section is targetted at explaining how to use the debugger from any version of Pydev. If you want more details on how to use the remote debugger, please check the Remote Debugger page.
