This manual assumes that you have already installed Eclipse and Pydev Extensions. If you have still not done so, please go to the download area and follow its instructions. Python and/or Jython installed in your machine is also a 'must'.


You can verify if it is correctly installed going to the menu 'help > about > plug in details' and checking if there are at least 5 plugins starting with 'com.python.pydev' listed with the correct version.


If at any time you wish to stop using the Pydev extensions plugin (or any other Eclipse plugin), you can disable it by going to the menu 'help > software updates > manage configuration', selecting the plugin and clicking 'disable'.
