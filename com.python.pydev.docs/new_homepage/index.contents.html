'Pydev Extensions' is a plugin allows Python editing in Eclipse. It works on top of Eclipse and Pydev, and adds features such as:

  • Context-Insensitive code completion with auto-import
  • Static code analysis, replacing PyLint in Pydev
  • Quick-Fix with auto-import for undefined tokens found in code analysis
  • Find definition replacement for Bycicle Repair Man
  • Debug server (to allow debugging scripts not lauched from within Eclipse)
  • Keywords presented as auto-completions as you type

The complete feature list may be found here.

You may download the plugin and evaluate it with full functionalities for one month without requiring a license.
