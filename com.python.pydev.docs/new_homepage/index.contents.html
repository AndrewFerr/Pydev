'Pydev Extensions' is a plugin allows Python editing in Eclipse. It works with Eclipse and the Pydev "Open Source" version, and adds features such as:

  • Context-Insensitive code completion with auto-import
  • Builtin code analysis, acting as a PyLint replacement (but much faster)
  • Quick-Fix for undefined tokens (suggesting auto-imports) found in code analysis
  • Quick-Fix to ignore other error messages
  • Find definition replacement for Bycicle Repair Man
  • Debug server (to allow debugging scripts not lauched from within Eclipse)
  • Keywords presented as auto-completions as you type

The complete feature list may be found here.

You can download the plugin and evaluate it with full functionalities for one month without requiring a license. The only difference between the licensed and the unlicensed version is that you'll get a message every 30 minutes saying that it is an unlicensed version (you have to license it to remove that message).
