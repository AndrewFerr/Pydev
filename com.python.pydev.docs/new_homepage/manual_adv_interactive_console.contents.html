Interactive Console

In Pydev Extensions, you can use Ctrl+Enter to open a console for you, and you can use the editor to pass commands to the opened console.

NOTE: This is only available for Pydev Extensions. In the open source version, you can have a python console for experimenting following the FAQ: How do I configure a python shell for experimenting with Python?

The Pydev editor enables you to pass commands directly from the editor to the console, either evaluating on each new line, or only on request for a given selected code (configured in the preferences -- see below).

Alternatively you may code directly in the Console if you want.

You may also choose some 'common' operations you want it to execute each time you create a new shell (configured in the preferences -- see below).
