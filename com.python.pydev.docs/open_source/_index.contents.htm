Some Background

PyDev has Fabio Zadrozny as its maintainer since October 2004. It has been created by Aleks Totic.

What is PyDev?

PyDev is a plugin that enables users to use Eclipse for Python and Jython development. It comes with many goodies such as code completion, syntax highlighting, syntax analysis, refactor, debug and many others. If you want more details on the provided features, you can check here.

PyDev Extensions

Pydev now has a new 'partner': Pydev Extensions. It works with Pydev, and delivers a number of features that can make it a 'unique' environment, providing (among other features) code-analysis (does not depend on external features and is much faster), code completion with auto-import, remote debugging, better go to definition, 'quick-fixes'....

If you want more details, check the Pydev Extensions homepage
