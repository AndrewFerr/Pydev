Sponsoring

Well, PyDev is provided as an Open Source Project, but without some kind of support, it is really hard to keep any project going, so, if you really like it and use it, you can help by donating. Higher donations can receive some space in the homepage (contact me through my user in sourceforge: fabioz if you have interest).

Also, custom projects can be negociated. So, if you really want something that is not currently implemented in PyDev, you can contact me.
