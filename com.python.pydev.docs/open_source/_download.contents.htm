    • Requirements:

  • Eclipse 3.1 (the SDK is required)
  • Python 2.3.x or 2.4.x
  • Java 1.4 or higher

    • Instructions:

    Fast install: just go to the update manager (inside the help menu) and add update site: http://pydev.sf.net/updates/ (eclipse should do the rest)

    An alternative is just getting the zip file and extracting it yourself in eclipse. If you choose to do it, just make sure the plugins folder is extracted on top of the eclipse plugins folder.

    Important:

    If you have version 0.9.6 or earlier installed, you must remove it and all previous versions before installing the latest version.
