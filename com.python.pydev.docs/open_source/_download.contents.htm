Current release: LAST_VERSION_TAG.

  • Use the Eclipse update manager: http://pydev.sourceforge.net/updates/
  • Get zip from SourceForge

    • Requirements:

  • Eclipse 3.2 (the SDK is required)
  • Python 2.3.x or 2.4.x
  • Java 1.4 or higher

    • Fast Install:

    Go to the update manager (inside the help menu) and add update site: http://pydev.sourceforge.net/updates/ (eclipse should do the rest)

    An alternative is just getting the zip file and extracting it yourself in eclipse. If you choose to do it, just make sure the plugins folder is extracted on top of the eclipse plugins folder.

    Complete Install:

    There is a 'complete install guide' at http://www.fabioz.com/pydev/manual_101_root.html so, if you have any problems in the install, that's the place you should check (it also guides you through configuring pydev correctly).

    Important:

    If you have version 0.9.6 or earlier installed, you must remove it and all previous versions before installing the latest version.
