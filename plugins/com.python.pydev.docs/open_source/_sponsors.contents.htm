Pydev Sponsors



  • Pydev Extensions is the commercial side of Pydev, and enables Pydev to keep growing. It aims at providing new functionalities to Pydev, such as code analysis, code completion with auto import, etc (check http://www.fabioz.com/pydev for more details). It also passes new technologies to the Pydev 'Open Source' version from time to time.





  • ESSS: Engineering Simulation and Scientific Software is a company that develops scientific software. It creates customized software (for scientific visualization and simulation) for many engineering purposes, and makes consulting in engineering-related fields.
