Current release: LAST_VERSION_TAG.

  • Urls to use when updating with the Eclipse update manager:
    • 1.4.3 onwards: http://pydev.sourceforge.net/updates/
    • Before 1.4.3: http://pydev.sourceforge.net/updates_old/
    • Nightly builds: http://nightly.aptana.com/pydev/site.xml

  • Get zip releases: SourceForge download

Requirements:

  • Eclipse 3.2, 3.3 or 3.4 (python requires only the platform release, and jython also requires JDT)
  • Python and/or Jython
  • Java 1.4 or higher

Fast Install:

Go to the update manager (inside the help menu) and add update site: http://pydev.sourceforge.net/updates/ or some other specified url (Eclipse should do the rest)

An alternative is just getting the zip file and extracting it yourself in eclipse. If you choose to do it, just make sure the plugins folder is extracted on top of the eclipse plugins folder and restart with '-clean'.

Complete Install:

There is a 'complete install guide' at http://www.fabioz.com/pydev/manual_101_root.html so, if you have any problems in the install, that's the place you should check (it also guides you through configuring pydev correctly).

Important:

If you have version 0.9.6 or earlier installed, you must remove it and all previous versions before installing the latest version.
