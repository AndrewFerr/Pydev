What is Pydev?

Pydev is a plugin that enables users to use Eclipse for Python, Jython and IronPython development, making Eclipse a first class Python IDE.

It comes with many goodies such as:

  • code completion
  • code completion with auto import
  • syntax highlighting
  • syntax analysis
  • code analysis
  • go to definition
  • refactor
  • mark occurrences
  • debug
  • tokens browser
  • interactive console
  • and many others:

For more details on the provided features, check the Features Matrix.

Release 1.5.0

Pydev Extensions is now Open Source!

What happened to Pydev Extensions?

Pydev Extensions is now merged with Pydev, and its once closed source code has become open source. Thus, there is no more Pydev Extensions, only the open source Pydev, with all the capabilities of Pydev Extensions incorporated.
