What is Pydev?

Pydev is a plugin that enables users to use Eclipse for Python, Jython and Iron Python development -- making Eclipse a first class Python IDE -- It comes with many goodies such as code completion, syntax highlighting, syntax analysis, refactor, debug, code analysis, tokens browser, local renames, interactive console and many others. If you want more details on the provided features, you can check Features Matrix.
