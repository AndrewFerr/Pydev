Pydev and Pydev Extensions are now products of Aptana, makers of the popular Aptana Studio, Eclipse-based IDE for Ajax, Jaxer, Ruby on Rails, and PHP. You can plug Pydev into Aptana Studio or both into Eclipse and use them side by side. Aptana plans deeper integrations in the future. See the news.


Why was 'Pydev Extensions' created?

Pydev Extensions was created with the idea of making a better environment to the current state of python programming.

It is not Open Source, so that more focused support can be provided for it (and it helps on fostering the Open Source version). Also, it was created to be a 'professional version' -- it is (currently) priced around the 'personal' version of other commercial IDEs, but, don't be fooled by that, it can surely match the best IDEs around (but please, don't take believe on that... go check each version and take your own conclusions).


What will happen to the Pydev 'Open Source' version?

Pydev Extensions was created with the idea of 'adding value' to the current 'Open Source' version, and not replacing it, thus, keeping the Pydev 'Open Source' version 'open source' is a priority.

Currently, there are plans to pass technology from the Pydev Extensions to the 'Open Source' version from time-to-time, altough it is still not very clear how will that be done, as that will depend a lot on how things go with the Pydev Extensions version.
