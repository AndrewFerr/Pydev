Go to definition

KEYBINDING: F3

The 'go to definition' actions enables you to get to a given definition. It is included in both distributions, in the Pydev 'Open Source' and in the Pydev Extensions, however, there are some differences among them:

The Pydev 'Open Source' version includes this feature based on BRM (Bycicle Repair Man), and Pydev Extensions implements it itself, with a number of advantages:

  • It is much faster
  • It works well on 'self' tokens
  • It can work on methods / attributes from parameters (as the image below shows).
