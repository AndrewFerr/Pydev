This manual has the following areas:

  • Getting started. This section covers the basics you need to know in order to get up and running with Pydev Extensions (it also covers the topics needed to get the Pydev "Open Source" version up and running).

  • Feature matrix and 'howto' for each feature. Read this if you want to know what features Pydev has or how to use a specific feature.

  • Keybindings:This is a reference chart with some of the available keybindings that may prove useful to you (I suggest you print it and leave it at your side until you learn how to master them...).

  • Screencasts: Want to see Pydev in action? Screencasts are probably a good idea...

  • Articles: This is where articles about Pydev will stay. Its first addition is a tutorial on how to script Pydev with Jython.

  • FAQ: The FAQ is still under construction, so, please refer to the Pydev 'Open Source' FAQ. You can use the same instructions to report bugs, add feature requests and check for doubts that appear in that FAQ.
