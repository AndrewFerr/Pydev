Pydev is a complete python development environment for eclipse (I guess you could also call it a Python IDE).


Intro

Pydev is currently maintained by Fabio Zadrozny - it has been created by Aleks Totic as a "getting to know Eclipse project".

It's been some time since no releases for pydev are done... This is due to a complete rewrite of code completion, so, plans have changed for release 1.0... It will only get to 1.0 after a stable version of code completion is created (right now, it depends on a Python shell that executes code for doing code completion). I added a screenshot of what I've gotten so far, so you know what to expect!


Notes for release 0.8.0

This release should have fixed the reported bugs on code completion and on markers (thanks for the feedback!)
And for those of you that had problems, please check them and see if it works!

Builders have also been added, so, now we can have TODO tasks and get Pylint to analyze python files.


Quick Summary

Pydev has still not reached 1.0, but it is already stable enough for python editing, with features such as:

  • Code Completion (disabled by default - see notes on the preferences page).
  • Refactoring with bicycle repair man.
  • Outline
  • Go to definition with F3 (powered by bicycle repair man)
  • Debugger
  • Content Assistant (Ctrl+1)
  • Code Coverage
  • Etc. (more details can be found at the features page)
