A little history

Pydev has Fabio Zadrozny as its 'project manager' (and as lead developer, software tester, etc).
Aleks Totic has started it as a "getting to know Eclipse project".

Like PyDev? Use it a lot? Want it to get even better?...

Then, help it achieve that!

You can help by donating (paypal) - 'software development' is something that really takes time, so, in short, this would help me manage the project better, mostly because I could dedicate more time to the project if I could get some money from it.

Or you could make patches, check it (or the homepage) for bugs, or simply go to the forum and answer some questions... Any kind of support is appreciated.

News:

New Release: 0.9.2!!

Well, as you all know, software management and development is something that appears almost out-of-control at times, so, many places had unit-tests added as I started to refactor and improve some areas, so, expect the quality to keep up, even while new features are added!

  • Scott Schleiser inside the editor... now on apply, it applies new settings, without restarting the editor.
  • Scott Schleiser inside the debugger... most 'implement me' and 'volunteers needed' were fixed.
  • New Content Assistants added.
  • Docstrings in national encodings should work.
  • from foo import bar, xxx, yyy... should work.
  • Custom colors for decorators and numbers.
  • Matching brackets highlighted.
  • Bugs: code-formatting and others...
