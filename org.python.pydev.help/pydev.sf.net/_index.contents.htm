Some Background

PyDev has Fabio Zadrozny as its maintainer. It has been created by Aleks Totic.

What is PyDev?

PyDev is a plugin that enables users to use Eclipse for Python development. It comes with many goodies such as code completion, syntax highlighting, syntax analysis, refactor, debug and many others. If you want more details on the provided features, you can check here.

New Release: 0.9.7

This build fixes some nasty bugs from 0.9.6... it is highly recommended you install it. (right now, everything but the debugger should work with 3.1).

Important notes

Note 1: Because of some issues with Eclipse, you have to uninstall all previous PyDev installations before installing the new release.
Note 2: This release only works with Eclipse 3.1 (and from now on, Eclipse 3.0.x is no longer supported).
