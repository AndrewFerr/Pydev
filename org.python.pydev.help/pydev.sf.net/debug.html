Pydev debugger

Debugger, one step behind the editor. The ultimate goal is a nice graphical debugger, but until then here is a small improvement to the External Tools method:

There is a Python context menu in Navigator:

If you pick "Run...", you'll get the standard Eclipse debug dialog:

Here you can pick your program arguments, etc. The dialog will try to verify that your python interpreter is present. If it is not, go to Preferences:Pydev:Debug and configure a valid python executable.

And when you run the program, the output is in the console. The errors in the console are hyperlinked back to the file:
