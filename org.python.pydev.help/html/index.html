Pydev guide

Pydev is a Python development environment for Eclipse. It is an open-source one-man effort so far.

The current release is 0.3. The editor is stable, and debugger is very much work in progress.

  • syntax highlighting
  • syntax analysis, first synax error gets marked
  • works with tabs or spaces.
  • outline view that shows functions, imports, classes
  • you can run a python script from the navigator context menu.
  • errors in console output get hyperlinked to its file locations
  • update site at http://pydev.sourceforge.net/updates/

Projects home page is at: http://pydev.sourceforge.net/
