Pydev manual

Pydev is a Python development environment for Eclipse. It is an open-source effort, hosted at http://pydev.sourceforge.net/. See the website for the up-to-date development information.

Pydev consists of two main components, the editor and the debugger. The editor is stable, and provides the following features:

  • syntax highlighting
  • syntax analysis, first synax error gets marked
  • works with tabs or spaces.
  • outline view that shows functions, imports, classes
  • limited hyperlinks: by pressing ctrl key you can jump to functons defined inside the same file.

This is the first public release of the debugger, so its stability is unknown. The debugger provider the following features:

  • fully integrated into Eclipse's debugging framework.
  • execution control: you can set breakpoints, step in/out/over
  • supports python threads.
  • variables/stack are there, just like other IDE's
  • console traces are hyperlinked to files. Even files outside of workspace can be opened through hyperlinks.

Some of the advanced Eclipse features are not there: conditional breakpoints, code execution.
