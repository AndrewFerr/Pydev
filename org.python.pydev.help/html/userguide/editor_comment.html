In order to make bulk comments, highlight a body of text in the editor, and do one of the following:

  • Source Menu -> Comment
  • Right-Click -> Source -> Comment
  • Ctrl+/
You may remove a selected bulk comment with:
  • Source -> Comment
  • Right-Click -> Source -> Uncomment
  • Ctrl+\
